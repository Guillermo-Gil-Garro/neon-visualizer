# Neon Visualizer

Interactive web-based neon sign visualizer.

Upload SVG designs, apply realistic neon glow effects, preview on different backgrounds, reposition elements, edit individual vector paths, and export high-quality PNG previews.

## Features
- SVG upload and automatic color detection
- Realistic neon glow rendering
- Custom neon color override
- Background selection and custom background upload
- Drag & keyboard positioning
- Element selection and per-path color editing
- PNG export

## Usage
Open `index.html` directly in your browser or serve the project using a local server (recommended).

## Notes
When using external background images, PNG export may be limited by browser CORS restrictions.  
For guaranteed export, use local background images served from the same origin.
